## 5. Content Optimization Checklist {#optimization-checklist}

### Pre-Publish Checklist (Every Article)

#### Title & Meta
- [ ] Primary keyword in title (first 60 characters)
- [ ] Title is compelling and click-worthy
- [ ] Meta description 150-160 characters
- [ ] Meta description includes primary keyword
- [ ] Meta description has a call-to-action
- [ ] URL slug is short and keyword-rich

#### Content Structure
- [ ] H1 tag contains primary keyword
- [ ] Primary keyword in first 100 words
- [ ] H2 tags with secondary keywords (3-5)
- [ ] H3 tags for sub-sections
- [ ] Content length 1,500-2,500 words
- [ ] Short paragraphs (2-3 sentences)
- [ ] Bullet points and numbered lists
- [ ] Clear section headings

#### Keyword Optimization
- [ ] Primary keyword density 1-2%
- [ ] Secondary keywords naturally included
- [ ] LSI keywords scattered throughout
- [ ] No keyword stuffing

#### Internal Linking
- [ ] 3-5 internal links per article
- [ ] Links to relevant tool pages
- [ ] Links to related blog posts
- [ ] Descriptive anchor text

#### External Linking
- [ ] 1-2 external links to authoritative sources
- [ ] Links open in new tab
- [ ] Links are relevant and helpful

#### Images
- [ ] At least 2-3 images per article
- [ ] Images compressed for fast loading
- [ ] Alt tags with descriptive text
- [ ] Image file names include keywords
- [ ] Relevant screenshots or diagrams

#### Technical SEO
- [ ] Schema markup (Article, FAQ)
- [ ] Mobile responsive design
- [ ] Page load under 3 seconds
- [ ] Proper heading hierarchy

#### Engagement
- [ ] Clear call-to-action
- [ ] Social sharing buttons
- [ ] Related posts section
- [ ] Comments enabled

#### Promotion
- [ ] Share on social media (3 platforms minimum)
- [ ] Send to email list
- [ ] Post in relevant communities
- [ ] Submit to aggregators

---

### Tool Page Optimization Checklist

#### SEO
- [ ] Title tag with primary keyword
- [ ] Meta description 150-160 characters
- [ ] H1 with tool name
- [ ] H2 sections for features and how-to
- [ ] FAQ section with schema markup

#### User Experience
- [ ] Clear tool description
- [ ] Step-by-step instructions
- [ ] Example/demo included
- [ ] Error handling messages
- [ ] Download/share functionality

#### Conversion
- [ ] Clear primary CTA
- [ ] Related tools section
- [ ] Blog post links
- [ ] Social proof elements
